Struct pe_util::PE

source · 
#[repr(transparent)]
pub struct PE<'a> { /* private fields */ }
Expand description

A pointer sized type that allows the user to read a buffer in memory as a Windows PE. Currently only supports 32-bit and 64-bit PEs on x86 architectures, but, I plan on supporting more architectures in the future. This type is indifferent to 32 or 64-bit architecture, as well as whether the file has been mapped into memory for execution, or if the file is still in its on disk stat.

Example

fn example(slice: &[u8]) {
    let pe = PE::from_slice(slice).expect("Could not validate that slice is a valid PE file.");
    let exports = pe.get_exports().unwrap_or_default();

    for export in exports {
        let ord = pe.get_function_ordinal(export.as_bytes());
        println!("Export: {export} Ordinal: {ord}");
    }

    let res = pe.get_pe_resource(10, 100).expect("Could not find PE resource");

    println!("{}", res.len());
}
